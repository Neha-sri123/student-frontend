# Student Frontend Project

This is a frontend web application developed using React and Vite.

## Description
This project provides a student frontend interface with fast performance and modern design.

## Technologies Used
- React
- Vite
- JavaScript
- HTML
- CSS

## How to Run

Install dependencies:
npm install

Run the project:
npm run dev

Open in browser:
http://localhost:5173
